Three.js Journey Challenge February

Developer: 2blue1clown

Theme: Solar System


I have been learning Three.js for a about a month now and this is my first submission to the challenge. Thank you for building such a great community Bruno!


For this challenge I decided to create representation of the solar system using data from windows2universe.org. The model is built using each planet's

  • size (logarithmic scale)
  • orbital semi-major axis (logarithmic scale)
  • orbital eccentricity
  • orbit's inclination to the ecliptic
  • orbit's period
  • tilt relative to its orbit
  • rotational period

I was super suprised to learn that the planets closer to the sun have longer days than Earth!
