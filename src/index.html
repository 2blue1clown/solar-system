Three.js Journey Challenge February

Theme: Solar System

Author: 2blue1clown

For this challenge I decided to create a physcially accurite representation of the solar system. Data from windows2universe.org is used to make a scale* representation of the solar system with elliptical orbits.

*To make the model easily viewable to the user the sizes of the celestial bodies are placed on a logarithmic scale. So is the size of the eliptical orbits.
